## 二、根因分析：为什么 F7 在大 N 时失败

### 2.1 组件分解（N=20）

```
Family      │ logH │ -λΣ_hist │ wall  │ F7   │ logH rank → F7 rank
─────────────┼──────┼──────────┼───────┼──────┼───────────────────
Lor5D        │ 33.5 │ -2.1     │ 0.00  │ 31.5 │ 15 → 1
KR_2layer    │ 33.8 │ -1.5     │ 0.00  │ 32.3 │ 16 → 2  ← 与Lor5D几乎相同！
Lor4D        │ 32.6 │ -2.3     │ 2.15  │ 32.4 │ 14 → 3
Lor2D        │ 21.0 │ -4.6     │ 16.00 │ 33.3 │  1 → 4  ← wall在拯救低logH
RLk6_mid     │ 22.9 │ -4.2     │ 16.00 │ 35.3 │  3 → 6  ← 低logH+满wall
KR_like      │ 27.0 │ -2.2     │ 15.84 │ 43.3 │ 11 → 17 ← wall惩罚成功
```

### 2.2 结构性矛盾

1. **logH 增长 O(N)**：Lor4D/5D 因果结构稀疏 → 线性扩展多 → logH 最高
2. **wall 衰减 O(N^{-0.5})**：α_N = 16·(20/N)^{0.5}，N=36 时仅 ~11.9
3. **layered 结构两头占便宜**：低 logH（层内约束多）+ 高 R（满 wall）
4. **KR_2layer 是致命反例**：R ≈ 0, wall = 0, 高 logH — 与 Lor5D 不可区分

### 2.3 不可调和的对立

```
                低 logH        高 logH
    ┌──────────────┬──────────────┐
高R │ RLk6 等       │ TransPerc    │  ← wall 惩罚有效
    │ (F7中等)      │ (F7最高)     │
    ├──────────────┼──────────────┤
低R │ (不存在)      │ Lor4D/5D     │  ← wall 无效
    │              │ KR_2layer    │  ← 无法与Lor区分！
    └──────────────┴──────────────┘
```

## 三、文献综述核心结论

### 3.1 Kleitman-Rothschild (1975)
> 几乎所有 n 元偏序集都是三层结构：n/4, n/2, n/4。

- 这就是我们的 KR_like 家族
- Lorentzian 撒点在所有偏序集中是**指数稀少**的
- 我们的比较是：F7 能否从指数多的 KR 中选出指数少的 Lorentzian？

### 3.2 Dhar (1978)
> 定义 S(ρ) = lim 2n⁻² ln N(n,ρ)，其中 ρ = 可比对比例。

- **ρ 正是我们的 f₂** = C₀/C(N,2)
- S(ρ) 在 ρ₁ ≤ ρ ≤ ρ₂ 有平台 = (ln2)/2
- 在 ρ₁ 处发生一阶相变（KR 开始主导）
- Dhar 将问题映射为有三体相互作用的格点气体

### 3.3 Prömel-Steger-Taraz (2001)
> 证明了随 ρ 增加有无穷多个相变。

- ρ ~ 1/4: KR 三层主导
- ρ 更高: 两层、四层、多层结构依次出现
- Carlip 所说"KR 之后最常见的是两层和四层"正是此结论

## 四、logH ≠ 物理熵

### CST 路径积分 vs 线性扩展

| 量 | 含义 | 对应物理 |
|----|------|---------|
| Z = Σ_C exp(iS_BDG[C]) | 对不同因果**结构**求和 | 配分函数 |
| L(P) = 线性扩展数 | 对同一结构的不同**标签**求和 | 无直接对应 |

### 最佳物理解释
- **观测熵**：观测者可赋予的时间排序数
- **Boltzmann 计数**：标签=微观态，因果结构=宏观态
- **但不是**爱因斯坦方程中的引力熵（∝ 面积）

### 结论
Carlip 正确：logH 在 CST 中没有确立的物理基础。我们将其用作"复杂度"代理，
但未从第一性原理推导出它与时空选择的关系。

## 五、诚实评估

### ✅ 有效的部分
- Sigmoid wall 在小 N 正确识别 Lorentzian vs KR
- Σ_hist 捕捉有意义的结构差异
- d_eff/f₂ 分析（Prediction A）是扎实的数学
- 四个推论 (A/B/C/D) 的内部一致性

### ❌ 失败的部分
- F7 在 N≥28 时被 Carlip 的对照组击败
- logH 没有确立的物理理由
- 家族选择从 Dhar/Prömel 角度看是 cherry-picking
- 框架未与 CST 路径积分严格连接

### ⚠️ 关键发现
**KR_2layer 是致命反例**：它与 Lor5D 在 F7 的所有组件上几乎相同
（R≈0, wall≈0, 高logH, 低Σ_hist），说明 F7 缺少区分两者的物理自由度。

## 六、前进路径

### 方向 A：用 S_BDG 替代 logH
- 优点：S_BDG 直接连接 Einstein-Hilbert 作用量
- 缺点：bd_d4 实验（§4.1.23）已显示 S_BDG 区分度差

### 方向 B：ρ 条件采样对比（Dhar 框架）
- 固定 ρ = f₂(4) ≈ 0.05，在该密度下采样所有偏序集
- 与 Lor4D 对比：这是 Dhar/Prömel 的正确比较框架
- 问题：如何在固定 ρ 下均匀采样偏序集？（MCMC 方法）

### 方向 C：信息几何方法
- 用 (ρ, Σ_hist, d_eff, ...) 参数化偏序集
- 展示 Lor4D 在几何不变量上的独特性
- 不依赖 logH，改用纯几何量构建泛函
