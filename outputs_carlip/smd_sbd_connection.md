## 2. Decomposing S_MD Features in C_k Basis

### 2.1 Feature ↔ Interval Count Mapping

The three S_MD features are functions of {C_k}:

1. **d_eff**: monotone function of R = C_total/N(N-1)/2
   - R = f₂(d_eff), where f₂(d) = Γ(d+1)Γ(d/2)/(4Γ(3d/2))
   - So d_eff is determined by C_total = Σ_k C_k
   - δ_d = d_eff - d*  ∝  (R - R*) / f₂'(4)  ∝  (Σ C_k - Σ* C_k)

2. **C₁/C₀**: ratio of first two interval counts
   - δ_c = C₁/C₀ - c*  = (C₁ - c*·C₀) / C₀

3. **width_ratio**: not directly a function of {C_k}
   - But empirically correlated with interval structure
   - Test below

### 2.2 Width as function of interval counts

**N=48**: width ~ β₀·C₀/n² + β₁·C₁/n² + β₂·C₂/n² + β₃·C₃/n² + β₄·R + β₅

## 3. Formal Mapping: S_MD ↔ S_BD

### 3.1 S_BD in normalized form

S_BD^(4)/N = 1 - C₀/N + 9C₁/N - 16C₂/N + 8C₃/N

Define normalized interval densities: c̃_k = C_k / [N(N-1)/2]
Then S_BD/N = 1 + (N-1)/2 · (-c̃₀ + 9c̃₁ - 16c̃₂ + 8c̃₃)

### 3.2 S_MD in C_k form

Feature vector components:
  δ_d = f₂⁻¹(R) - 4    where R = Σ c̃_k
  δ_c = C₁/C₀ - c*(N)
  δ_w = w - w*(N)

S_MD = Σ_{ij} Σ⁻¹_{ij} δ_i δ_j

Key structural difference:
- S_BD = **linear** in C_k → first-order functional of poset
- S_MD = **quadratic** in (δ_d, δ_c, δ_w) → second-order functional

### 3.3 Taylor expansion bridge

Near the 4D Lorentzian fixed point, write C_k = C_k^* + ΔC_k.

Then S_BD = S_BD^* + Σ_k c_k ΔC_k    (exact, since S_BD is linear)

And S_MD = Σ_{ij} Σ⁻¹_{ij} [∂I_i/∂C_k · ΔC_k] [∂I_j/∂C_l · ΔC_l]

        = ΔC^T J^T Σ⁻¹ J ΔC

where J_{ik} = ∂I_i/∂C_k is the Jacobian.

So **S_MD = ΔC^T · M · ΔC** where M = J^T Σ⁻¹ J

and **S_BD = S_BD^* + c^T · ΔC** where c = BDG coefficients.

The connection: S_BD fixes a **hyperplane** in C_k-space,
while S_MD defines an **ellipsoid**. They are complementary:

- S_BD=S_BD^* defines the locus of posets with correct average curvature
- S_MD=0 defines the locus of posets indistinguishable from Lor4D

## 7. Summary: S_MD ↔ S_BD Connection

### 7.1 Structural Relationship

Both S_MD and S_BD are functionals of poset interval structure {C_k}:

$$S_{\mathrm{BD}} = \sum_k c_k C_k \quad (\text{linear in } C_k)$$

$$S_{\mathrm{MD}} = \Delta\mathbf{C}^\top \mathbf{J}^\top \Sigma^{-1} \mathbf{J} \, \Delta\mathbf{C} \quad (\text{quadratic in } \Delta C_k)$$

where J is the Jacobian ∂(d_eff, C₁/C₀, w)/∂(C₀, C₁, C₂, C₃).

### 7.2 Complementary Roles

| Property | S_BD | S_MD |
|----------|:----:|:----:|
| Order in C_k | Linear (1st) | Quadratic (2nd) |
| Physical meaning | Scalar curvature | Structural identification |
| Selects | Correct dynamics | Correct geometry |
| Lor4D status | One of many minima | **Unique** minimum |
| Free parameters | d-dependent c_k | 0 (Mahalanobis) |
| N→∞ behavior | Fixed ∝ R | Diverges for non-4D |

### 7.3 Key Finding

S_BD and S_MD are **complementary, not redundant**:
- S_BD fixes average curvature (dynamics)
- S_MD fixes geometric identity (structure)
- Their intersection {S_BD ≈ flat} ∩ {S_MD ≈ 0} = Lor4D

This supports the structural existence interpretation:
the path integral weight exp(iS_BD) favors certain curvatures,
while S_MD certifies that the resulting poset actually looks like
a 4D Lorentzian manifold. The two actions operate at different levels
of the same interval counting hierarchy.
