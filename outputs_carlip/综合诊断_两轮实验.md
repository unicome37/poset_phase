### 第一轮：F7 → 17 家族（已知结果回顾）
- F7 在 N≥28 时 Lor4D 排名下滑至 #8–#11

### 第二轮-A：S_link 替代 logH（本次新实验 1）

**结论：更差。**

| N | F_link1 (纯) | F_link3 (+wall) |

| 16 | Lor4D #13 | Lor4D #3 |

| 28 | Lor4D #7 | Lor4D #5 |
| 36 | Lor4D #8 | Lor4D #5 |
| 48 | Lor4D #8 | Lor4D #5 |

**根因**：KR/layered 结构的 link density 天然高于 Lorentzian sprinklings。
S_link = N - 2C₀ 惩罚低链密度结构——而 Lor4D **就是**低链密度的。

链密度排名（N=48）：
```
AbsLayer  0.333  ← 最密
KR_like   0.253
KR_4layer 0.233
KR_2layer 0.196
Lor3D     0.145
Lor4D     0.137  ← 比所有 KR 都稀疏
Lor5D     0.093
```

**结论**：BDG link action 作为 Lor vs KR 的**区分器**不可行。
（注意：这与"BDG 在路径积分中抑制 KR"不矛盾——后者是在 KR 内部用
BDG 的**交替正负系数**产生抑制效果，不是用链密度排序。）

### 第二轮-B：信息几何——17 家族特征空间（本次新实验 2）

**结论：突破性发现！**

#### 发现 1：KR_2layer 在 (d_eff, R) 平面与 Lor4D 不可区分

```
Lor4D:     d_eff = 3.93, R = 0.234  ← 高维低R
KR_2layer: d_eff = 3.87, R = 0.000  ← 也在"Lor角"！
```

这解释了为什么 **所有基于 d_eff 或 R 的泛函都失败**：
- F7 的 wall 基于 R → KR_2layer 的 R=0 让 wall 失明
- F10 的 d_eff-well → KR_2layer 的 d_eff≈3.87 落在 well 里

#### 发现 2：扩展到 12 维特征空间后，Lor4D 与 KR_2layer **清晰可分**

N=48 时 10/12 特征达到 |Δ|/σ > 2.0 的分离度：

|------|:-----:|:---------:|:------:|---------|
| **width_ratio** | 0.410 | 0.749 | **10.78** | Lor 宽度分布均匀，KR 一层独大 |
| **C₁/C₀** | 0.195 | 0.000 | **5.61** | Lor 有非平凡区间，KR 全是链 |
| **σ_hist** | 0.117 | 0.062 | **5.54** | Lor 层级更深 |
| **R** | 0.234 | 0.000 | **5.44** | Lor 有非链因果对 |
| **chain_ratio** | 0.062 | 0.042 | **3.87** | Lor 最长链更长 |
| **S_d4/N** | 0.886 | -3.465 | **3.83** | BDG d=4 作用量完全不同 |
| **S_link/N** | -5.46 | -7.93 | **3.19** | 链密度差异 |

#### 发现 3：分离度随 N 增长**单调增强**

所有可分特征的效应量从 N=20 到 N=48 **单调增长**：

```
width_ratio:  3.76 → 8.10 → 10.78  ✅ 趋势正确
C₁/C₀:       1.53 → 3.56 → 5.61   ✅ 趋势正确  
R:            1.68 → 4.23 → 5.44   ✅ 趋势正确
σ_hist:       2.09 → 4.62 → 5.54   ✅ 趋势正确
```

这是**决定性的好消息**：Lor4D 与 KR_2layer 在大 N 极限下**不是**趋同的，
而是**趋异**的。旧 F7 的失败不是因为"两者真的一样"，
而是因为 F7 选错了看它们的"眼睛"（logH + R + d_eff）。

#### 发现 4：区分 Lor4D 和 KR_2layer 的关键是**区间结构**

KR_2layer 的独特病理：
- **所有因果关系都是链（C₁=C₂=C₃=0）**：每对可比元素之间没有中间元素
- **宽度极大（width_ratio ≈ 0.75）**：一层包含 N/2 个元素
- **链极短（chain_ratio ≈ 0.04）**：只有 2 层

Lor4D 的独特签名：
- **非平凡区间（C₁/C₀ ≈ 0.20）**：有丰富的因果中间结构
- **适度宽度（width_ratio ≈ 0.41）**：空间截面合理
- **深链（chain_ratio ≈ 0.06）**：时间方向有层级深度

**物理翻译**：Lor4D 是一个因果钻石——有丰富的因果中间结构和适度的
时空切片宽度。KR_2layer 是一个退化的"全链"结构——每对可比元素之间
没有因果中间者。尽管两者的 d_eff 和 f₂ 几乎相同，
**区间形状（interval shape distribution）**完全不同。

### 第三轮：LSD 原型——单调奖励失败（本次新实验 3）

用信息几何发现的 killer features 构建 Lorentzian Structural Discriminator：

| 变体 | 架构 | 得分 | N=64 排名 | KR_2layer? |
|------|------|:----:|:---------:|:----------:|
| LSD1 | -α·(C₁/C₀) - β·(1-w) | 20.5% | #12 | ❌ 全败 |
| **LSD2** | LSD1 + wall(R) | **89.7%** | #10 | ❌ N≥48 |
| LSD3 | LSD2 + chain_ratio | 83.3% | #13 | ❌ N≥48 |
| LSD4 | interval_richness + wall | 79.5% | #14 | ❌ N≥48 |

**关键教训**：单调奖励（"更多 C₁/C₀ 越好"）失败，因为：
- Layered 结构的 C₁/C₀ **远大于** Lor4D（0.6–1.0 vs 0.20）
- KR_2layer 的 C₁/C₀ = 0，但 R=0 让 wall 失明
- Wall 衰减问题与 F7 完全相同

**结论**：需要 **非单调 Well 架构**——不是"越多越好"，而是"恰好在 Lor4D 的值附近最好"。

#### Well 中心物理分析（第五轮实验）

**Q1: d\* ≈ 3.93 → 第一原理可导 ✅**
- 理论预测：d\* = 4.0（时空维数）
- 实测 d_eff(Lor4D) 在 N=20–64 范围内 ≈ 3.79–4.01，Δ% = 3.2%
- d\*=3.93 是有限 N Myrheim-Meyer 偏差，N→∞ 时 d\*→4

**Q2: c\* ≈ 0.213 (C₁/C₀) → 有解析公式但 N-dependent ⚠️**
- 解析：C₁/C₀ = E[Nu·e^{-Nu}]/E[e^{-Nu}], u ~ Beta(2,2)
- 实测从 0.10 (N=20) → 0.26 (N=64)，**漂移 63%**
- c\*(N) 是 N 的函数，不是常数！
- 但可解析计算 → 不是自由参数，而是 **导出函数**

**Q3: w\* ≈ 0.408 (width_ratio) → 部分可导，N-dependent ⚠️**
- 实测从 0.53 (N=16) → 0.35 (N=64)，**漂移 34%**
- 理论来源：因果钻石空间截面体积剖面
- 精确公式需要 Alexandrov interval 完整体积积分

**结论**：LSD-W2 可重写为
```
F = α·(C₁/C₀ − c_theory(4,N))² + β·(w − w_theory(4,N))² + γ·(d_eff − 4)²
```
其中 c_theory 和 w_theory 是 d=4 Minkowski sprinkling 的解析预测。
**唯一真正的自由参数是 α, β, γ（相对权重）**。

## 战略结论

### ❌ 已关闭的路径

1. **F7 (logH + wall)** → logH 增长压过 wall，且 logH 无物理基础
2. **F10 (logH + d_eff-well)** → d_eff(KR_2layer) ≈ d_eff(Lor4D)，well 失明
3. **S_link 替代 logH** → KR 链密度天然高于 Lor
4. **任何基于 (d_eff, R) 二元组的泛函** → KR_2layer 在此平面与 Lor4D 不可分
5. **LSD 单调奖励** → Layered 的 C₁/C₀ >> Lor4D，单调奖励偏好 Layered

### ✅ 已验证的成功路径

**LSD-Well：在区间形状 × 横向结构 × 维度的特征空间中构建 Well 势**

```
F_LSD-W2 = 5·(C₁/C₀ − 0.213)² + 5·(width − 0.408)² + 2·(d_eff − 3.93)²
```

- Lor4D **#1/17 at ALL N** (16–64) ✅
- Beats **ALL** KR-family, layered, and other structures ✅
- KR_2layer margin **grows with N** (+22% → +600%) ✅
- **不用 logH**，纯因果几何量 ✅
- 回应 Carlip 三条批评 (C1 ✅, C2 ✅, C3 ✅)

### 🔍 遗留问题

1. **Well 中心是外部输入** — c*, w*, d* 来自 Lor4D 实测均值
2. 需要从第一原理推导 well 中心（可能需要 EH 路径积分论证）
3. 仅在 LSD-W1（无 d_eff-well）下得分 97.4% — 说明 C₁/C₀ + width 已是 97% 的信息

Conjecture E 的 §4.1.25 已经发现：
- p_k = C_k/ΣC_k 的 PC2+（shape residual）携带曲率信息（11.4%）

- **C₁/C₀ 正是 shape 通道的一个投影**

区分 Lor4D 与 KR_2layer 的特征（区间形状），
