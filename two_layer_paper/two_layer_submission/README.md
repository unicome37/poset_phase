# Two-Layer Screening Submission Package

Target journal:
- `Physical Review D`

Package contents:
- `two_layer_screening.tex` - submission version of the main manuscript
- `two_layer_screening.pdf` - compiled PDF for quick review
- `cover_letter.tex` - PRD cover letter draft
- `arxiv_metadata_draft.md` - arXiv metadata draft (title/authors/category/comments/abstract)
- `arxiv_submission_checklist.md` - arXiv upload checklist
- `figures/` - local figure copies for standalone submission-folder compilation

Notes:

  two-layer screening architecture.
- The `Prediction E` material is now only referenced lightly as external support
  and is developed in a separate submission package.
- Remaining warnings are layout-level only (mainly overfull hbox); the PDF compiles successfully.
